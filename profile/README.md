## porcelain

git proxy for your entire dev toolchain — [porcelain.sh](https://porcelain.sh)

`git fetch` your issues, PRs, reviews, and comments. porcelain exposes your dev platforms as queryable shadow branches, so project metadata and code arrive in a single `git fetch` instead of a dozen separate API calls. push to a shadow branch and the change syncs back to the source — GitHub today, with Linear, Sentry, and deploy platforms next. access control and audit trails come from git's native versioning.
